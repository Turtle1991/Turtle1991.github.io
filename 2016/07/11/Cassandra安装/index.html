Cassandra安装

  • Cassandra

最近项目中有用到ca，所有决定开始学学，这次是在虚拟机下安装个单机版本，做个记录。

下载安装包

wget http://mirrors.cnnic.cn/apache/cassandra/3.7/apache-cassandra-3.7-bin.tar.gz
tar -zxvf apache-cassandra-3.7-bin.tar.gz -C /usr/cassandra-3.7

创建需要的目录

cd /usr/cassandra-3.7
data/commitlog、data/data、data/saved_caches、logs

启动

[root@newBlash log]# cassandra -f -R //用 -R 选项，因为默认不建议使用root用户来启动

运行 cqlsh

需要 python2.7，centos 升级 python2.6 到 2.7，之前有安装过2.7版本，所以直接

mv /usr/bin/python /usr/bin/python_old //备份旧的 2.6版本
cp /usr/local/python2.7/bin/python2.7 /usr/bin/python //复制新的 2.7版本

检查： python -V //提示 2.7

修改 yum，因为只支持 2.6版本

vim /usr/bin/yum

 #!/usr/bin/python2.6 //指明是用2.6版本的python

到这里就替换成功，目前没有发现问题。

其他

1.Cassandra官网的start：http://wiki.apache.org/cassandra/GettingStarted

2.Cassandra的配置文件cassandra.yaml部分参数说明：

默认集群名（cluster_name）：Test Cluster
默认缓存文件存放路径（saved_caches_directory）：$CASSANDRA_HOME/data/saved_caches
默认数据文件存放路径（data_file_directories）：$CASSANDRA_HOME/data/data
默认commitlog存放路径（commitlog_directory）：$CASSANDRA_HOME/data/commitlog
默认的Cassandra服务地址（rpc_address）：localhost
默认的Cassandra服务端口（rpc_port）：9160
默认的CQL本地服务端口（native_transport_port）： 9042
